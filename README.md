# PomoFlow

Vanilla JavaScript ile hazirlanmis kucuk bir Pomodoro ve gorev takip sitesi.

## Neler var?

- Pomodoro ve mola zamanlayicisi
- Loop video arka plan destegi
- Spotify sarki, album ya da playlist oynaticisi
- Ayarlanabilir sureler
- Gorev ekleme, tamamlama ve silme
- Tarayicida yerel kayit
- Mobil ve masaustu uyumlu arayuz

## Calistirma

`index.html` dosyasini tarayicida acman yeterli.

## Video arka plan

Loop olarak kullanmak istedigin videoyu su dosya adiyla ekle:

```txt
assets/background.mp4
```

Spotify icin uygulamadaki kutuya Spotify sarki, album ya da playlist linki yapistirabilirsin.

## Kontrol

```bash
npm run check
```
